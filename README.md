###应用场景
当别人开始给你提bug的时候 你可能一天要向同样的repo同样的branch提n个pull request。
用这个工具可以节省时间。

###原理
gitlab提供非常友好的URL，可以用URL指定branch repo提pullrequest.

###用法
在gitlab网站，创建一个merge request，点compare branch，观察下URL。你懂的。
我个人就不关这个程序了，没弄数据持久。
